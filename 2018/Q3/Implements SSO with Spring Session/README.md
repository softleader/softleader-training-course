docker 執行 redis

```docker run --rm -d -p 6379:6379 redis```

找 redis 的 container id

```docker ps```

進入redis中

```docker exec -it (containerId) redis-cli```

找所有在redis中的key

```keys *```

清除所有在redis中的key

```flushdb```

示範url

```localhost:8181/index/ERIC```
